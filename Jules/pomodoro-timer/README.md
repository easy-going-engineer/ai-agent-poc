# Pomodoro Timer

ポモドーロ・テクニックをサポートする多機能タイマーアプリです。  
シンプルな操作で集中作業と休憩を繰り返し、作業効率を高めることができます。

## 主な特徴

- **3つのモード**  
  ポモドーロ（作業）・短い休憩・長い休憩をワンクリックで切り替え可能
- **時間設定**  
  各モードの時間を自由に設定可能
- **タイマー操作**  
  開始・一時停止・リセットボタンで直感的に操作
- **履歴管理**  
  セッション終了ごとに履歴を自動記録（最大20件、ブラウザに保存）
- **外観カスタマイズ**  
  背景色や画像を自由に設定可能
- **レスポンシブデザイン**  
  Bootstrap採用でPC・スマホ両対応

## 使い方

1. `src/index.html` をブラウザで開きます（インストール不要）。
2. タイマー表示下のボタンで「開始」「一時停止」「リセット」ができます。
3. 「Pomodoro」「Short Break」「Long Break」ボタンでモードを切り替えます。
4. 「Settings」セクションで各モードの時間を設定し、「Save Settings」で保存します。
5. 「Customize Appearance」で背景色や画像URLを設定し、「Apply Appearance」で反映します。
6. セッションが終了すると履歴に自動記録されます（最大20件まで表示）。

※設定・履歴・外観はブラウザのlocalStorageに保存されます。同じPC・ブラウザでのみ保持されます。

## 注意事項

- 本アプリはlocalStorageを利用しています。履歴や設定は同一ブラウザ・同一PCでのみ有効です。
- 外部ライブラリとして[Bootstrap 5](https://getbootstrap.com/)をCDN経由で利用しています。
- モバイル端末にも対応しています。
